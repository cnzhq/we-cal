# 📊 CSV 日历编辑器使用指南

通过 Excel 编辑 CSV 文件，转换为 JSON，再生成 ICS 日历。

---

## 🚀 快速开始

### 1. 编辑 CSV 文件

在 `csv/` 文件夹中创建或编辑 `.csv` 文件：

**示例：`csv/苏超2026常规赛程.csv`**

```csv
周次,场次,日期,开球时间,持续小时,主队,客队,比赛场地,区县,地级市,转播
1,1,4月11日,1935,2,常州,南通,常州奥体中心体育场,新北区,常州市,江苏省城市足球联赛 微信视频号/抖音/微博视频号/百家号
1,2,4月11日,1940,2,扬州,苏州,扬州体育中心体育场,邗江区,扬州市,江苏省城市足球联赛 微信视频号/抖音/微博视频号/百家号
```

### 2. CSV → JSON

```bash
npm run convert
```

### 3. （可选）检查/修改 JSON

查看 `events/苏超2026常规赛程.json`，手动调整需要修改的内容。

### 4. JSON → ICS

```bash
npm run build
```

### 5. 推送到 GitHub

```bash
git add .
git commit -m "更新赛程"
git push origin main
```

## 📋 CSV 字段说明

| 列名 | 必填 | 说明 | 示例 |
|------|------|------|------|
| `周次` | 否 | 比赛周次 | `1` |
| `场次` | 否 | 总场次序号 | `1` |
| `日期` | **是** | 中文日期 | `4月11日` |
| `开球时间` | 否 | 24小时制 | `1935` (19:35) |
| `持续小时` | 否 | 比赛时长 | `2` (默认2小时) |
| `主队` | **是** | 主队名称 | `常州` |
| `客队` | **是** | 客队名称 | `南通` |
| `比赛场地` | 否 | 场地名称 | `常州奥体中心体育场` |
| `区县` | 否 | 区/县 | `新北区` |
| `地级市` | 否 | 所属城市 | `常州市` |
| `转播` | 否 | 转播信息 | `微信视频号/抖音...` |

## 💡 高级功能

### 智能合并

如果 `events/` 中已存在同名 JSON 文件，转换时会：
- 保留 CSV 中解析的事件
- 保留 JSON 中手动添加的事件（不重复）
- 保留 JSON 中的日历名称、时区等元数据

### 年份自动识别

文件名中包含年份会自动识别：
- `苏超2026常规赛程.csv` → 使用 2026 年
- `比赛2025.csv` → 使用 2025 年
- `无年份.csv` → 使用当前年份

### 字段别名支持

CSV 支持多种列名写法：

| 标准列名 | 替代写法 |
|---------|---------|
| `周次` | `周`, `week` |
| `场次` | `场`, `match` |
| `日期` | `date` |
| `开球时间` | `时间`, `time` |
| `持续小时` | `时长`, `duration` |
| `主队` | `主场`, `home` |
| `客队` | `客场`, `away` |
| `比赛场地` | `场地`, `venue` |
| `区县` | `区`, `district` |
| `地级市` | `城市`, `city` |
| `转播` | `直播`, `broadcast` |

## ⚠️ 注意事项

1. **文件编码**：保存 CSV 时使用 UTF-8 编码（Excel 默认通常是 UTF-8）
2. **日期格式**：使用 `M月D日` 格式，如 `4月11日`
3. **时间格式**：使用 4 位数字，如 `1935` 表示 19:35
4. **空行处理**：CSV 中的空行会被自动跳过
5. **中文逗号**：如果字段内容包含逗号，建议用 Excel 编辑（自动处理引号）

## 🎯 完整示例

**CSV 输入：**
```csv
周次,场次,日期,开球时间,持续小时,主队,客队,比赛场地,区县,地级市,转播
1,1,4月11日,1935,2,常州,南通,常州奥体中心体育场,新北区,常州市,江苏足球联赛直播
```

**生成的 JSON：**
```json
{
  "calendarName": "苏超赛程",
  "timezone": "Asia/Shanghai",
  "description": "2026年苏超完整赛程...",
  "events": [
    {
      "title": "⚽ 常州 vs 南通",
      "start": "2026-04-11T19:35:00",
      "duration": "PT2H",
      "description": "苏超第1周（总第1场）：常州队主场。转播：江苏足球联赛直播",
      "location": "常州奥体中心体育场，新北区，常州市",
      "reminder": 30
    }
  ]
}
```

**生成的 ICS：** 可直接导入日历软件订阅
